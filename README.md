# React VOD PWA

Very simple mock video podcast Progressive Web App, built with React/Redux    

## install

```
$ yarn install
```

## Run webpack-dev-server with simple node app running on different port    

```
$ yarn start
```

## build prod (webpack)

```
$ yarn deploy
```

## tests

```
# Run once
$ yarn test

# watch
$ yarn test:watch
```
